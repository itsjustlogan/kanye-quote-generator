This project uses the following technologies:

- React.js

## **Installation**

1. Clone project

```bash
git clone https://github.com/itsjustlogan/kanye-quote-generator.git
```

2. cd into directory

```bash
cd kanye-quote-generator
```

3. Install dependencies

```bash
npm i
```

## **Usage**

1. Start dev server

```bash
npm run dev
```
